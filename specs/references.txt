## Key Language Features
- **dbug.check()**:        REPOS/aria-docs/guide/debugging/dbug.md
  Assert-like conditional checks with grouped logging and custom failure actions.
  `dbug.check(group, fmt, data, condition, action?)` — triggers when condition is FALSE.
  Based on: REPOS/educational/dbug-utility/ (JS proof-of-concept)
- **dbug.print()**:        Same file — grouped debug logging to hex stream
- **failsafe**:            Required in every program — global error handler

## ABI Gotchas (see also /memories/repo/aria_language_gotchas.md)
- String params = `const char *`, NOT AriaString struct
- String returns = AriaString `{char*, int64_t}` by value
- flt32/flt64 both pass as `double` at C ABI
- "free" in extern names → shadowed, use `_release`/`_destroy`
- `failsafe` func required in every program
- Reserved words: raw, ok, stream, is, end, max
