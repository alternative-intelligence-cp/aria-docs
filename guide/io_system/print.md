# Print & Output

## print / println

```aria
print("no newline");
println("with newline");
println(42);                        // prints integer
println(`Value: &{expression}`);    // interpolation
```

- `print(s)` — output without newline
- `println(s)` — output with newline

## Streams

| Stream | Purpose |
|--------|---------|
| `stdout` | Standard text output |
| `stderr` | Error output |
| `stddbg` | Debug output |
